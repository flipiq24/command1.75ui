# FlipIQ UI/UX Prototype Documentation

## Document Purpose
This document serves as a comprehensive UI/UX reference for the FlipIQ real estate investment dashboard prototype. It details all user workflows, screen designs, interaction logic, conversation flows, and gamification elements. Use this for design handoffs, stakeholder reviews, and development reference.

# Part 1: The FlipIQ Machine â€” User Journey Overview

## 1.1 Core Concept
FlipIQ is a **guided workflow system** (the "machine") that keeps Acquisition Associates (AAs) focused on high-priority activities throughout their workday. The system:

- **Starts each session** with an AI check-in (iQ)
- **Guides users** through Deal Review and Daily Outreach
- **Tracks progress** via completion percentages
- **Celebrates milestones** with gamified animations
- **Allows re-entry** at any point without losing context
- **Reports activity** back to Acquisition Managers (AMs)

# Part 3: Gamification & Celebration System

## 3.1 Progress Tracking

### Completion Percentage Calculation

```
completionPercent = (
  (hotDealsCompleted / hotDealsTotal) +
  (warmDealsCompleted / warmDealsTotal) +
  (coldDealsCompleted / coldDealsTotal) +
  (dailyOfferGoalMet ? 1 : 0)
) / 4 * 100
```

### Progress Display Locations

| Location | Format |
|----------|--------|
| Action Plan | "You have completed X% of today's follow-ups" + progress bar |
| Sidebar | Counters on Deal Review and Daily Outreach links |

## 3.2 Milestone Celebration Modal

### Trigger Conditions

```
IF completionPercent >= 100
AND user clicks iQ button (on PIQ page next to Notes)
THEN trigger celebration
```
